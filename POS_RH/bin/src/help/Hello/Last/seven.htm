退货

    当顾客凭销售小票要求退货时，执行以下操作办理退货：
    【操作说明】
    第1步 按 退货 键；
    第2步 输入授权人用户名和密码；
    第3步 输入商品的退货数量；
    第4步 按 数量 键；
    第5步 扫描商品条码；当无法扫描时，直接输入商品条码或者油品热键 90# / 97# / 0# ；
    第7步 输入授权人用户名和密码；
    第8步 重复第1-7步，直至需要的退货的商品全部过机；
    第9步 按 0 键，再按 现金结算 键；
    第10步 按“待收”处显示的金额（负数）退回顾客金额；
    注意：
    油品不可退货。
